## Socket.IO Event Flow

### Task Movement Event

**Client Emits:**
```javascript
socket.emit('task-moved', {
  workspaceId: 'workspace_123',
  taskId: 'task_456',
  sourceColumnId: 'todo',
  destinationColumnId: 'in-progress',
  sourceIndex: 2,
  destinationIndex: 0
});
```

**Server Broadcasts:**
```javascript
socket.to(`workspace-${workspaceId}`).emit('task-moved', {
  taskId: 'task_456',
  sourceColumnId: 'todo',
  destinationColumnId: 'in-progress',
  sourceIndex: 2,
  destinationIndex: 0
});
```

**All Clients Receive:**
```javascript
socket.on('task-moved', (data) => {
  // Update Redux store
  dispatch(moveTaskRealtime(data));
  // UI updates automatically via React
});
```
